void ModbusTCP::idle ( void(*)()  idle)

Set idle time callback function (cooperative multitasking).

This function gets called in the idle time between transmission of data and response from Unit. Do not call functions that read from the serial buffer that is used by ModbusTCP. Use of i2c/TWI, 1-Wire, other serial ports, etc. is permitted within callback function.
